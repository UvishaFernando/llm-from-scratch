# LLM From Scratch

A hands-on journey to understand and build Large Language Models from scratch using PyTorch.

## Topics

* Self-Attention
* Multi-Head Attention
* Transformer Blocks
* Tokenization
* Training Tiny LLMs
* Fine-Tuning
* RLHF

This repository documents my learning process, implementations, notes, and experiments while studying modern LLM architectures.
